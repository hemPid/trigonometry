trigonometry.js is a small script which contains 6 trigonometric functions.
sin, cos, tg, asin, acos, atg (also known as arcsin, arccos, arctg).
All this functions are stored in object  called trig. So if you want to call some of this functions, you schould write trig[function_name](a)
For example:
trig.sin(30) //0.500
trig.cos(60) //0.500
trig.asin(0.5) //30.000
All functions accept only degree parameters and returns rounded to third decimal place value.




trigonometry.js -  это маленький скрипт, который содержит в себе 6 тригонометрических функций.
sin, cos, tg, asin, acos, atg (так же известные как arcsin, arccos, arctg).
Все эти функции хранятся в объекте trig и для того чтобы вызвать одну из этих функций надо использовать вот эту конструкцию trig.имя_функции(a)
Например:
trig.sin(30) //0.500
trig.cos(60) //0.500
trig.asin(0.5) //30.000
Все функции принимают параметры только в градусах и возвращают значение, округлённое до третьего знака после запятой.
